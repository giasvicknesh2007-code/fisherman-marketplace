# Fisherman Marketplace

A cloud-powered seafood marketplace web application where fishermen can post and manage fish listings in real time.

## Live Demo
https://coastal-fish-network-cfn.web.app

## Features
- Firebase Authentication
- Firestore Database
- Secure listing creation
- Delete own listings
- Persistent login sessions
- Responsive marketplace feed
- Image URL support
- Firebase Hosting deployment

## Tech Stack
- HTML
- CSS
- JavaScript
- Firebase Auth
- Firestore
- Firebase Hosting

## Project Structure
/public
  - index.html
  - style.css
  - app.js

## Deployment
Hosted on Firebase Hosting.
